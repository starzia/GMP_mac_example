How to make your Xcode project compile with GMP

These instructions show how I setup the test program in this repository.

Prerequisite: install GMP using Homebrew ("brew install gmp").

Step 1: double-click the project and edit the build phases:

Set some compiler flags for your source file. NOTE: please copy and paste "-lgmp -lgmpxx -I/usr/local/include" because the first two parameters use a lower-case "l" for "library" and the third parameter is an upper-case "I" for "include."

Step 2: add to "Link Binary with Libraries" and choose "Add Other.."

Step 3: type "Apple-Shift-G" to go to a hidden folder, and type "/usr/local/lib" because that is where Homebrew installed GMP.

Step 4: Open the "libgmp.dylib" and "libgmpxx.dylib" dynamic library files. You can also add these one at a time.
